# Backend API

This is the backend for the project, built using **Node.js** and **Express**.

## Setup

### 1. Install dependencies
```bash
npm install
```

### 2. Run the server
- In production:
  ```bash
  npm start
  ```
- In development (auto-restart on changes):
  ```bash
  npm run dev
  ```
